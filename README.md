# Gaggle Mockup

[Live Demo Here](https://pardamike.github.io/gaggle-mockup/)

## Notes
- Using Angular Material
- Has lazy loaded modules (`guest` and `dashboard` modules)
- Multiple layouts (`guest` and `authenticated`)
- No external dependencies other than Angular and Angular Material (no Bootstrap, Angular Flexbox, Foundation, etc)

## Login and Registration
To log in, simply fill out the username and password with annything, then click log in.

To register, simply fill out the registration form, submit, and you will be "logged in".

## Installation Steps
- Clone `git clone https://github.com/pardamike/gaggle-mockup.git`
- Install `npm install`
- Run `ng serve --open`
